A book review app for Ethiopic books similar to goodreads. This is going to be a niche app focusing for this kind of market.

### Features

* Book catalog

  * Title, Author, Genre, Language, Publisher, Year, ISBN (if available)

  * Cover upload

  * Description / summary field

  * “Submitted by” metadata (for community-contributed books)

* Search & Browse

  * Filter by author, language, genre, or year

  * Simple full-text search

* Giving review on books

  * Logged-in users can leave a text review + star rating (1–5)

  * Show average rating and number of review

* Rating a book using a star system

* Crowd sourcing book database by allowing users to submit book records

* Want to read list

* Book club and discussion about books with members
